# WebGL 연습용 프로젝트



#### About

React로 WebGL을 다루는 방법을 연습하는 개인 프로젝트입니다.

웹팩과 바벨을 사용하여 프로젝트를 구성하였습니다.

타입스크립트를 사용합니다.
